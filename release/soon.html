chuck-1.2.0.0 is the next major release of ChucK, scheduled for January February March April May As Soon As It's Ready 2005.

what's new:
  • objects, building unit generators directly in the language
  • arrays
  • events
  • language specification
  • build system (thanks stacken!)
  • better on-the-fly programming support
  • better documentation (thanks adam!)
In the meantime:

  • join the mailing list for announcement, updates, discussions, and tutorials.
  • download chuck-1.1.x.x, which demonstrates the fundamental ChucK properties of timing, concurrency, and on-the-fly programming.
