ezNote

inherits : Object

Basic SMucK note object carrying information about the note's onset, duration, pitch, and velocity. To be used in score importing, generating, editing, and playback.

constructors

ezNote()

Default constructor for ezNote. Initializes with default values.

ezNote(float onset, float beats, int pitch, int velocity)

Construct an ezNote with the given parameters.

member variables

float onset()

The onset of the note relative to the start of the measure, in beats. Default value of 0.0

float beats()

The duration of the note in beats. Default value of 1.0

int pitch()

The pitch of the note, as MIDI note number. Default value of 60

int velocity()

The velocity of the note, as a MIDI velocity value (0-127). Default value of 100

int[] CC()

MIDI CC values (CURRENTLY NOT IMPLEMENTED).

Set the duration of the note in beats.

Set the pitch of the note.

Set the velocity of the note.

ezMeasure

inherits : Object

SMucK measure object. A measure is a collection of ezNotes. Contents can be set using the SMucKish input syntax. Multiple measures can be combined into an ezPart.

constructors

ezMeasure()

Default constructor for ezMeasure. Creates empty measure.

ezMeasure(string input)

Create an ezMeasure from a SMucKish input string.

ezMeasure(string input, int fill_mode)

Create an ezMeasure from a SMucKish input string. The fill_mode parameter is used to specify how ezNote parameters are inferred. 0 = when a parameter is not specified, it is assigned a default value. 1 = when a parameter is not specified, it is assigned a value from the previously parsed note.

member variables

ezNote[] notes()

The ezNote objects in the measure.

float length()

The duration of the measure in beats.

float onset()

The onset of the measure relative to the start of the score, in beats.

member functions

void set_pitches(string input, int fill_mode)

Set the pitches of the notes in the measure using a SMucKish input string. The fill_mode parameter is used to specify how ezNote parameters are inferred (e.g. if number of provided rhythms is different than number of provided pitches). 0 = when a parameter is not specified, it is assigned a default value. 1 = when a parameter is not specified, it is assigned a value from the previously parsed note.
