ezNote

inherits : Object

Basic SMucK note object carrying information about the note's onset, duration, pitch, and velocity. To be used in score importing, generating, editing, and playback.

constructors

ezNote()

Default constructor for ezNote. Initializes with default values.

ezNote(float onset, float beats, int pitch, int velocity)

Construct an ezNote with the given parameters.

member variables

float onset()

The onset of the note relative to the start of the measure, in beats. Default value of 0.0

float beats()

The duration of the note in beats. Default value of 1.0

int pitch()

The pitch of the note, as MIDI note number. Default value of 60

int velocity()

The velocity of the note, as a MIDI velocity value (0-127). Default value of 100

member functions

void set_onset(float onset)

Set the onset of the note.

void set_beats(float beats)

Set the duration of the note in beats.

void set_pitch(int pitch)

Set the pitch of the note.

void set_velocity(int velocity)

Set the velocity of the note.

ezScore

inherits : Object

SMucK score object. An ezScore object contains one or more ezParts. Score contents can be set using the SMucKish input syntax, or by importing a MIDI file. ezScore objects can be passed to an ezScorePlayer object for playback.

constructors

ezScore()

Default constructor for ezScore. Creates empty score.

ezPart(string input)

Create an ezScore from a SMucKish input string. Puts contents into a single ezPart with a single ezMeasure. If given a MIDI filepath (ending in '.mid'), creates the score from the provided MIDI file.

ezPart(string filename, float bpm)

Create an ezScore from a MIDI file (ending in '.mid'), with a given tempo in beats per minute.

member variables

float bpm()

The tempo of the score in beats per minute. Default value of 120.

int time_sig_numerator()

The numerator of the time signature. Default value of 4. Used mostly with MIDI files to infer measure boundaries.

int time_sig_denominator()

The denominator of the time signature. Default value of 4. Used mostly with MIDI files to infer measure boundaries.

member functions

void addPart(ezPart part)

Add an ezPart to the score.

float getScoreEnd()

Get the end time of the score in beats (the last note's release point).

float getScoreDuration()

Get the duration of the score in ms.

float getMaxPolyphony(int part)

Get the maximum number of simultaneous notes the given part number.

void importMIDI(string filename)

Import a MIDI file. Each track in the MIDI file is added as a separate ezPart with a single ezMeasure.

void numParts(int num)

Set the pitches of the notes using a SMucKish input string. Puts contents into a single ezPart with a single ezMeasure.

Set the pitches of the notes using an array of SMucKish string tokens. Puts contents into a single ezPart with a single ezMeasure.

void set_pitches(int input[][])

Set the pitches of the notes directly using a 2D array of ints. Puts contents into a single ezPart with a single ezMeasure.
