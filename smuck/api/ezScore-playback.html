Objects used to play back ezScore score data.

ezScorePlayer

inherits : Object

Object used to play back ezScore objects using ezInstrument objects. Initialize an ezScorePlayer with an ezScore object and then assign ezInstruments to individual parts.

constructors

ezScorePlayer()

Default constructor for ezScorePlayer. Creates empty score player.

ezScorePlayer(ezScore score)

Construct an ezScorePlayer with the given ezScore object.

member variables

ezScore score()

The ezScore object to be played back.

ezParts[] parts()

The ezPart objects associated with the score. Set automatically if player is initialized with an ezScore.

ezInstruments[] instruments()

The ezInstruments to be used in playback. Need to set manually for each part.

int loop()

Whether to loop playback upon reaching end of score. Default value of 0 (false).

float rate()

The playback rate. Default value of 1.0. Used to speed up or slow down playback. Negative values play the score in reverse.

member functions

void setInstruments(ezInstruments @ instruments[])

Set the ezInstruments to be used in playback for all parts.

void play()

Start playback.

void pause()

Pause playback.

void stop()

Stop playback.

Move the playhead to the given position in time.

void pos(float beatPosition)

Move the playhead to the given position in beats.
