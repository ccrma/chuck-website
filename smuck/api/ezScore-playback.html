ezScorePlayer

inherits : Object

Object used to play back ezScore objects using ezInstrument objects. Initialize an ezScorePlayer with an ezScore object and then assign ezInstruments to individual parts.

constructors

ezScorePlayer()

Default constructor for ezScorePlayer. Creates empty score player.

ezScorePlayer(ezScore score)

Construct an ezScorePlayer with the given ezScore object.

member variables

ezInstruments[] instruments()

The ezInstruments to be used in playback. Need to set manually for each part.

int loop()

Whether to loop playback upon reaching end of score. Default value of 0 (false).

ezParts[] parts()

The ezPart objects associated with the score. Set automatically if player is initialized with an ezScore.

float rate()

The playback rate. Default value of 1.0. Used to speed up or slow down playback. Negative values play the score in reverse.

ezScore score()

The ezScore object to be played back.

member functions

void pause()

Pause playback.

void play()

Start playback.

void pos(dur timePosition)

Move the playhead to the given position in time.

void pos(float beatPosition)

Move the playhead to the given position in beats.

void preview()

Preview the score. Plays the score back with a default "preview instrument" using SinOscs.

void setInstruments(int partIndex, ezInstruments @ instrument)

Set the ezInstrument to be used in playback for a given part.

void setInstruments(ezInstruments @ instruments[])

Set the ezInstruments to be used in playback for all parts.

void setScore(ezScore score)

Set the ezScore object to be played back.

void stop()

Stop playback.

ezInstrument

inherits : Chugraph : UGen : Object

Base class for user-defined instruments to be used for playback in ezScorePlayer.

member variables

int n_voices()

The number of voices in the instrument.

member functions

void noteOn(ezNote theNote, int voice)

This function defines behavior that will be executed when a note is played by the ezScorePlayer. Typically, this would include setting frequency, gain, calling .keyOn(), etc. by referencing variables of theNote. The base class function is empty, and should be overridden by the user.

void noteOff(int voice)

This function defines behavior that will be executed when a note is released by the ezScorePlayer. This could include calling .keyOff(), etc. or referencing variables of theNote. The base class function is empty, and should be overridden by the user.
