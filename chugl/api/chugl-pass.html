RenderPass

inherits : GPass : SG_Component : Object

Render pass for drawing a GScene. If RenderPass.scene() is not set, will default to the main scene, GG.scene() If RenderPass.colorOutput() is not set, will default to the screen. If RenderPass.camera() is not set, will default to the scene's main camera: GG.scene().camera()

examples

  • basic/hud.ck

constructors

RenderPass()

Default constructor for RenderPass.

member functions

void autoClearColor(int clear)

Set whether the framebuffer's color target should be cleared each frame. Default true.

int autoClearColor()

Get whether the framebuffer's color target is cleared each frame.

void camera(GCamera camera)

Set the camera to use for rendering the scene. Defaults to the main camera of the target scene. You can call .camera(null) to use the scene's main camera.

GCamera camera()

Get the camera used for rendering the scene. If not set, will default to the scene's main camera.

void colorOutput(Texture color_texture)

Set the target texture to draw the scene to.

Texture colorOutput()

Get the target texture to draw the scene to.

void msaa(int sample_count)

Set the MSAA sample count for the render pass (default 4)

int msaa()

Get the MSAA sample count for the render pass.

void resolution(int width, int height)

Set the resolution of the output framebuffer. Default 0x0. If either dimension is set to 0, will auto-resize to the window size.

void scene(GScene scene)

Set the scene to render. If not set, will default to the main scene, GG.scene()

GScene scene()

Get the scene this pass is rendering.
