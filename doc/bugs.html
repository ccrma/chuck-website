ChucK : Bugs!

version: 1.4.x.x (numchucks)


If you find a bug, or experience a crash, please send us the CK file that caused it (or a minimal version that exhibits the bug), your operating system and version, and your ChucK version number.

Your ChucK version number can be found either through "miniAudicle -> About" in miniAudicle, or by the following command line invocation:

%> chuck --about

Please report at the ChucK github repo, send it to one of the developers or post it to a users/dev mailing list or forum.

Happy ChucKing, we hope!
