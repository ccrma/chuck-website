Overview:

ChucK is a strongly-typed, strongly-timed, concurrent audio and multimedia programming language. It is compiled into chuck bytecode instructions, which is then run in the ChucK Virtual Machine. Language documenation, API reference, and examples can be found here. This guide documents additional usage options .
